수업 제목에서 드러나듯이 본 수업에서는 large-scale 소프트웨어를 개발하는 경험을 쌓게 됩니다. 여러 가지 기술과 소프트웨어 시스템의 복잡도를 핸들링하는 법을 배웁니다.

4인 1조의 팀으로 협업하여 coding conventions, specifications, software design, software architecture, testing, abstraction, modularity, design patterns, software development process, version control, code review 등 소프트웨어 개발의 원리에 대해 몸소 느끼며 실전 지식을 습득할 수 있습니다. 하나의 웹 서비스를 차근차근 만들어나가는 과정에서 오는 성취감은 이루 말할 수 없을 정도입니다. 팀원들과 함께 하나의 스타트업을 운영한다는 생각으로 모두가 각자의 한계를 넘어가며 임합니다.

벌써 며칠 뒤면 최종 발표가 있습니다. 지난 한 학기를 돌아보면 정말 많은 것을 배웠던 것 같습니다. 컴퓨터공학부 학생이지만 개발 경험이라곤 콘솔로 하는 지뢰 찾기 정도밖에 없었던 제가 그새 이런 (나름 제겐) 엄청난 결과물을 만들어냈다는 것에 스스로가 대견하고 고생해주신 교수님과 조교님들께 감사할 따름입니다.
